Documentation

Complete guide to using AI Job Search for your job application workflow.

Installation

Option 1: Download Standalone

Download pre-built executables from the download page.

Option 2: Install from Source

# Clone the repository
git clone https://github.com/MadsLorentzen/ai-job-search.git
cd ai-job-search

# Install dependencies
bun install

# Build standalone executables
./build.sh

Option 3: Claude Code (Recommended)

Clone this repo and use it directly with Claude Code for the full AI experience:

git clone https://github.com/MadsLorentzen/ai-job-search.git
cd ai-job-search
claude

Setup

Run the setup command to configure your profile:

/setup

This will prompt you for:

  • Full name
  • Email address
  • Phone number
  • Location
  • Target job titles
  • Target locations
  • Skills and technologies
  • Experience level
Tip: Your profile is saved to data/profile.json and used to tailor all applications.

Configuration

Configuration files are stored in:

  • data/profile.json - Your candidate profile
  • data/privacy.json - Privacy settings
  • data/jobs/ - Scraped job postings
  • data/applications/ - Generated applications

FAQ

Do I need an API key?

AI Job Search works with Claude Code. The AI features require a Claude Code subscription or API key.

Which job portals are supported?

We support 12 job portals including LinkedIn, Indeed, Glassdoor, StepStone, Jobindex, Jobnet, and more.

Is my data private?

Yes! All data is stored locally on your machine. Nothing is sent to external servers except the job portals you search.

Troubleshooting

Executable won't run on Windows

Make sure to extract all files from the ZIP archive before running. Windows SmartScreen may block the first run - click "More info" then "Run anyway".

No jobs found

Try broadening your search criteria or check if the job portal is accessible from your location.
